# Translations

It is possible to add a translation for specific fields. The API supports translations in every [two-letter language ISO 639-1 code](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes), but UiTdatabank's main language is `nl` and the UiTdatabank UI only supports translations in `fr`, `de` and `en`.
